dotnet sln new    (Make a new solution).
dotnet sln "SolutionName" add "ProjectName.csproj" (add an existing project).
dotnet build (Build a project)
dotnet run (Run // //)
dotnet new -i MonoGame.Templates.CSharp::3.8.0.1641 (Install Monogame Templates).
dotnet new mgdesktopgl -o YourGameFolderName (Make a new DesktopGL project). 
cd YourGameFolderName (Change Directory)
dotnet tool install --global dotnet-mgcb --version 3.8.0.1641 (Install Monogame Content Builder).
dotnet tool install --global dotnet-mgcb-editor --version 3.8.0.1641 (// Content Editor).
mgcb-editor --register (Register Content).
mgcb-editor (Open MGCB Editor).
dotnet publish -r win-x64 -c Release (Publish Game to windows x64).
dotnet publish -r osx-x64 -c Release (// osx).
dotnet publish -r linux-x64 -c Release (// linux).
